# Documentation

New Tab replaces Obsidian's empty new-tab view with a customizable dashboard —
a background image, a clock, a greeting, quick search, recent files, bookmarks,
and a quote. This directory is the user-facing guide; for development and
contribution notes, see the [project README](../README.md).

## Overview

These pages take you from installing the plugin to tuning every widget:

- **New here?** Start with [Installation](installation.md), then [Usage](usage.md).
- **Customizing?** Every setting is documented in [Configuration](configuration.md).
- **Something not working?** See [Troubleshooting](troubleshooting.md).

## Documentation map

- [Installation](installation.md) — prerequisites, Community Plugins and BRAT installs, verifying, updating.
- [Configuration](configuration.md) — every setting, its default, and what it controls.
- [Usage](usage.md) — first use, common workflows, tips and shortcuts.
- [Troubleshooting](troubleshooting.md) — backgrounds or quotes not loading, debug logging, known issues.
